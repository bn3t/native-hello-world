# native-hello-world in a scoped package

> Native “Hello, world!” addon, useful for testing

## Installation

```console
$ npm install @bn3t/native-hello-world --save-dev
```

## Usage

```js
const helloWorld = require('@bn3t/native-hello-world')

helloWorld()
// => 'Hello, world!'
```
